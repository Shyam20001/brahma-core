# brahma-core

Rust + napi-rs native core for [Brahma-Firelight](https://github.com/Shyam20001/rsjs).  
Provides high-performance server bindings with prebuilt binaries for macOS, Linux, and Windows.

---

## Build

```bash
npm install
npm run build
```

## Test

```bash
npm run test # runs Vitest
```
